== Description ==

Versionmate gives you insight in your WordPress websites. Every website is provided with a risk factor based on the updates and vulnerabilities.

== Installation ==

1. Register your account at versionmate.com
2. Add your website and get an API key
3. Activate the Versionmate plugin
4. Insert your Versionmate API key into the Versionmate Settings page in your WordPress website
5. Done!
